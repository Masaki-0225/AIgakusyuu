## 画面要素

### ヘッダー

- アプリ名「MyTrelloアプリ１号」と今日の日付を表示する

### フィルターバー

ボード上部に表示する。選択中のボタンはハイライト表示する。

| ボタン | 動作 |
|--------|------|
| 全て | 全列（未着手・進行中・完了）を表示する（デフォルト） |
| 未着手 | 未着手列のみ表示する |
| 進行中 | 進行中列のみ表示する |
| 完了 | 完了列のみ表示する |

### 列（カラム）

- 「未着手」「進行中」「完了」の3列を横並びで固定表示する
- **カード追加ボタン（「＋ 追加」）は「未着手」列のみ**に設置する
  - 「進行中」「完了」への追加はカードを移動することで行う
- 各列のヘッダーには列名とカード枚数を表示する

### カード

| 表示項目 | 表示条件 |
|----------|----------|
| タイトル | 常に表示 |
| 期限日 | 入力されている場合のみ表示 |
| 優先度バッジ | 入力されている場合のみ表示（高=赤・中=黄・低=緑） |
| 完了日 | ステータスが done の場合のみ表示 |

期限日の色付け:
- 期限切れ（今日より前）: 赤背景
- 本日期限: 黄背景

### フォーム（カード追加・編集）

フォームはモーダルとして画面中央に表示する。

- タイトル入力欄（テキスト、必須）
- 説明文入力欄（テキストエリア、任意）
- 期限日入力欄（日付、任意）
- 優先度選択欄（高・中・低、任意）
- 保存ボタン（タイトルが空の場合は非活性） / キャンセルボタン

編集フォームには追加で:
- 削除ボタン（ゴミ箱アイコン）: クリック後に確認ダイアログを表示する

---

## 操作

- カードのドラッグ&ドロップによる**列間移動**に対応する
- カードクリックで編集フォーム（モーダル）を開く
- モーダルの背景クリックでモーダルを閉じる
