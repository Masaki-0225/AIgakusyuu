# 画面設計

## 画面要素

### ヘッダー

- アプリ名「MyTrelloアプリ１号」を表示する

### 列（カラム）

- 「未着手」「進行中」「完了」の3列を横並びで固定表示する
- **カード追加ボタン（「＋ 追加」）は「未着手」列のみ**に設置する
  - 「進行中」「完了」への追加はカードを移動することで行う
- 各列のヘッダーには列名・カード枚数の右隣に並び替えボタン（「優先度」「期日」）を表示する

### 並び替えボタン

各列のヘッダーに2つのボタンを配置する。

| ボタン | 動作 |
|--------|------|
| 優先度 | 優先度の高い順（高→中→低→なし）で並び替える |
| 期日 | 期日が近い順（昇順・期日なしは末尾）で並び替える |

- ボタンは押すたびにソート適用／解除をトグルする
- ソート適用中はボタンをハイライト表示する
- ソート中に手動でカードを並び替えると、そのカラムのソートは自動的に解除される

### カード

| 表示項目 | 表示条件 |
|----------|----------|
| タイトル | 常に表示 |
| 期限日 | 入力されている場合のみ表示 |
| 優先度バッジ | 入力されている場合のみ表示（高=赤・中=黄・低=緑） |
| 削除ボタン（✕） | 常に表示 |

### フォーム（カード追加・編集）

- タイトル入力欄（テキスト、必須）
- 説明文入力欄（テキストエリア、任意）
- 期限日入力欄（日付、任意）
- 優先度選択欄（高・中・低、任意）
- 保存ボタン / キャンセルボタン

---

## 操作

- カードのドラッグ&ドロップによる**列間移動**に対応する
- カードのドラッグ&ドロップによる**同一列内の並び替え**に対応する（ドラッグ中は挿入位置をラインで表示）
- カードクリックで編集フォームを開く
