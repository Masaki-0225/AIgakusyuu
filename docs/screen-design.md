# 画面設計

## 画面構成

```
┌──────────────────────────────────────────────┐
│           MyTrelloアプリ１号                  │  ← ヘッダー
├─────────────┬─────────────┬──────────────────┤
│   未着手    │   進行中    │      完了        │
│─────────────│─────────────│──────────────────│
│ ┌─────────┐ │ ┌─────────┐ │                  │
│ │タスクA  │ │ │タスクC  │ │                  │
│ │期限:6/15│ │ │優先度:高│ │                  │
│ └─────────┘ │ └─────────┘ │                  │
│ ┌─────────┐ │             │                  │
│ │タスクB  │ │             │                  │
│ └─────────┘ │             │                  │
│             │             │                  │
│  ＋ 追加   │  ＋ 追加   │  ＋ 追加        │
└─────────────┴─────────────┴──────────────────┘
```

---

## 画面要素

### ヘッダー

- アプリ名「MyTrelloアプリ１号」を表示する

### 列（カラム）

- 「未着手」「進行中」「完了」の3列を横並びで固定表示する
- 各列の下部に「＋ 追加」ボタンを設置する

### カード

| 表示項目 | 表示条件 |
|----------|----------|
| タイトル | 常に表示 |
| 期限日 | 入力されている場合のみ表示 |
| 優先度 | 入力されている場合のみ表示 |
| 削除ボタン | 常に表示 |

### フォーム（カード追加・編集）

- タイトル入力欄（テキスト、必須）
- 説明文入力欄（テキストエリア、任意）
- 期限日入力欄（日付、任意）
- 優先度選択欄（高・中・低、任意）
- 保存ボタン / キャンセルボタン

---

## 操作

- カードのドラッグ&ドロップによる列間移動に対応する
- カードクリックで編集フォームを開く
